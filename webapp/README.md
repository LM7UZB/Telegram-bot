# TillaBazar — Frontend (Telegram Mini App)

Bu papka faqat **frontend** (React + Vite). To'liq loyiha (bot + deploy) bo'yicha
asosiy yo'riqnomani ko'ring: [`../README.md`](../README.md).

## Lokal ishga tushirish
```bash
npm install
npm run dev      # http://localhost:3000
```

## Build
```bash
npm run build    # natija: dist/
```

## Muhit o'zgaruvchisi (ixtiyoriy)
`GEMINI_API_KEY` — AI yordamchisi uchun. `.env.example` ga qarang.
Kalit bo'lmasa ham ilova to'liq ishlaydi.
